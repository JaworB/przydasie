# Tiling and Window Management

Window navigation, layout control, and workspace management using SUPER key bindings.

## Focus Movement

| Binding | Action |
|---------|--------|
| `SUPER + LEFT` | Move focus left |
| `SUPER + RIGHT` | Move focus right |
| `SUPER + UP` | Move focus up |
| `SUPER + DOWN` | Move focus down |

## Window Operations

| Binding | Action |
|---------|--------|
| `SUPER + W` | Close window |
| `SUPER + J` | Toggle split (dwindle) |
| `SUPER + P` | Pseudo tiling |
| `SUPER + T` | Toggle floating/tiling |
| `SUPER + F` | Full screen (0) |
| `SUPER + CTRL + F` | Tiled full screen (0 2) |
| `SUPER + ALT + F` | Full width (1) |
| `SUPER + O` | Pop window out (float & pin) |

## Workspace Switching

| Binding | Action |
|---------|--------|
| `SUPER + [1-9, 0]` | Switch to workspace 1-10 |
| `SUPER + TAB` | Next workspace |
| `SUPER + SHIFT + TAB` | Previous workspace |
| `SUPER + CTRL + TAB` | Former workspace |
| `SUPER + mouse_scroll` | Scroll workspaces |

## Move Window to Workspace

| Binding | Action |
|---------|--------|
| `SUPER + SHIFT + [1-9, 0]` | Move window to workspace |
| `SUPER + SHIFT + ALT + [1-9, 0]` | Move silently (no focus) |

## Move Workspace to Monitor

| Binding | Action |
|---------|--------|
| `SUPER + SHIFT + ALT + LEFT` | Move workspace left |
| `SUPER + SHIFT + ALT + RIGHT` | Move workspace right |
| `SUPER + SHIFT + ALT + UP` | Move workspace up |
| `SUPER + SHIFT + ALT + DOWN` | Move workspace down |

## Window Swapping

| Binding | Action |
|---------|--------|
| `SUPER + SHIFT + LEFT` | Swap window left |
| `SUPER + SHIFT + RIGHT` | Swap window right |
| `SUPER + SHIFT + UP` | Swap window up |
| `SUPER + SHIFT + DOWN` | Swap window down |

## Resize Active Window

| Binding | Action |
|---------|--------|
| `SUPER + -` | Expand window left |
| `SUPER + =` | Shrink window left |
| `SUPER + SHIFT + -` | Shrink window up |
| `SUPER + SHIFT + =` | Expand window down |

## Scratchpad

| Binding | Action |
|---------|--------|
| `SUPER + S` | Toggle scratchpad |
| `SUPER + ALT + S` | Move window to scratchpad |

## Window Groups

| Binding | Action |
|---------|--------|
| `SUPER + G` | Toggle group |
| `SUPER + ALT + G` | Move out of group |
| `SUPER + ALT + LEFT/RIGHT/UP/DOWN` | Move to group |
| `SUPER + ALT + TAB` | Next in group |
| `SUPER + ALT + SHIFT + TAB` | Previous in group |
| `SUPER + CTRL + LEFT/RIGHT` | Group focus navigation |
| `SUPER + ALT + [1-5]` | Activate group window by number |

## Application Cycling

| Binding | Action |
|---------|--------|
| `ALT + TAB` | Next window |
| `ALT + SHIFT + TAB` | Previous window |

## Mouse Bindings

| Binding | Action |
|---------|--------|
| `SUPER + Left Click + Drag` | Move window |
| `SUPER + Right Click + Drag` | Resize window |
