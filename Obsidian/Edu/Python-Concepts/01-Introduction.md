# Introduction

Python is a versatile, interpreted programming language.

## What is Python?

- Easy to read and write
- Interpreted (no compilation)
- Dynamically typed
- Multi-paradigm (OOP, functional, procedural)

## Hello World

```python
# Console output
print("Hello, World!")
```

## Basic Syntax

```python
# Comments
# This is a comment

# Variables (no declaration needed)
name = "Alice"
age = 30
is_active = True

# Multiple assignment
a, b, c = 1, 2, 3

# Indentation (4 spaces recommended)
if age >= 18:
    print("Adult")
else:
    print("Minor")
```

## Python REPL

```bash
$ python3
>>> print("Hello")
Hello
>>> 2 + 2
4
>>> exit()
```
