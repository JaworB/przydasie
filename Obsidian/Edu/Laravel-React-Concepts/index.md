# Laravel + React Concepts

Full-stack web development learning path using Laravel (PHP) and React (JavaScript).

## Overview

- **Laravel**: Backend framework PHP
- **React**: Frontend library JavaScript
- **Stack**: Laravel as REST API + React as frontend
