# Introduction

JavaScript is a versatile, high-level programming language.

## What is JavaScript?

- Originally designed for web browsers
- Now runs on servers (Node.js), mobile apps, desktop apps
- Event-driven, object-oriented, functional

## ES6+ Features

| Feature | Description |
|---------|-------------|
| `let` | Block-scoped variable |
| `const` | Block-scoped constant |
| Arrow functions | Concise function syntax |
| Template literals | String interpolation |
| Destructuring | Extract values from objects/arrays |
| Spread operator | Expand arrays/objects |
| Classes | Syntactic sugar for prototypes |
| Promises | Handle async operations |

## Hello World

```javascript
// Console output
console.log("Hello, World!");

// DOM manipulation
document.getElementById("app").innerHTML = "Hello!";
```
