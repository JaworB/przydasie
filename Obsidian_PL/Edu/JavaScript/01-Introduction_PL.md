# Wprowadzenie

JavaScript to wszechstronny, wysokopoziomowy język programowania.

## Czym jest JavaScript?

- Oryginalnie zaprojektowany dla przeglądarek internetowych
- Teraz działa na serwerach (Node.js), aplikacjach mobilnych, desktopowych
- Sterowany zdarzeniami, obiektowy, funkcyjny

## Funkcje ES6+

| Funkcja | Opis |
|---------|------|
| `let` | Zmienna o zakresie blokowym |
| `const` | Stała o zakresie blokowym |
| Funkcje strzałkowe | Zwięzła składnia funkcji |
| Literały szablonowe | Interpolacja ciągów |
| Destrukturyzacja | Ekstrahowanie wartości z obiektów/tablic |
| Operator rozproszenia | Rozszerzanie tablic/obiektów |
| Klasy | Cukier składniowy dla prototypów |
| Promises | Obsługa operacji asynchronicznych |

## Hello World

```javascript
// Wyjście na konsolę
console.log("Witaj, Świecie!");

// Manipulacja DOM
document.getElementById("app").innerHTML = "Witaj!";
```
