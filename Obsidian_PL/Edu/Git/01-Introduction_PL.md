# Wprowadzenie

Git to **rozproszony system kontroli wersji**, który śledzi zmiany w kodzie źródłowym podczas tworzenia oprogramowania.

## Czym jest Git?

Git umożliwia wielu programistom współpracę, utrzymywanie historii i zarządzanie różnymi wersjami projektu.

## Kluczowe funkcje

- **Rozproszony**: Każdy programista ma pełną kopię repozytorium
- **Nieliniowy**: Wspiera równoległy rozwój przez gałęzie
- **Śledzenie zmian**: Rejestruje każdą modyfikację
- **Kopia zapasowa**: Pełna historia przechowywana lokalnie
- **Współpraca**: Zdalne serwery synchronizują zmiany
